# 🌐 Veldara — 웹에서 몰입형 3D 세계를 만들다

> **🔗 실행 링크**: [https://lgw7126.github.io/antigravity-landing-page-09](https://lgw7126.github.io/antigravity-landing-page-09)

---

## 📋 기획 개요

Svelte 5와 Three.js를 결합한 선언형 3D 웹 개발 라이브러리 **Veldara**의 공식 랜딩 페이지. 물리 엔진, 애니메이션, 모델 로더가 내장된 고성능 3D 웹 경험을 컴포넌트 기반으로 손쉽게 구성할 수 있음을 소개하는 페이지.

---

## ✨ 주요 특징

- **스크롤 비디오 배경** — Canvas 기반 스크롤 연동 영상 재생
- **SEO 최적화** — Open Graph, Twitter Card 메타태그 완비
- **Google Fonts** — Inter + Outfit 조합의 모던 타이포그래피
- **반응형 레이아웃** — 모바일/데스크톱 최적화

---

## 🛠️ 기술 스택

- **순수 HTML / CSS / JavaScript** (빌드 도구 없음)
- **Canvas API** — 스크롤 비디오 렌더링

---

## 🚀 실행 방법

별도 빌드 불필요. `index.html`을 브라우저로 직접 열거나, 로컬 서버를 사용:

```bash
# Python
python -m http.server 8080

# Node.js
npx serve .
```

[http://localhost:8080](http://localhost:8080) 접속
